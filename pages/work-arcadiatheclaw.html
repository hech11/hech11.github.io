Arcadia The Claw

  • Prev
  • Next
  • View Itch.io
  • View gameplay video

A game made for a college game jam. We were against 3 other teams and made this in 2 weeks. The goal is to escape the inside of a claw machine by collecting toy pieces that are randomly placed within the level. While finding the pieces the claw in the claw machine will keep grabbing toys and possibly the player, resulting in a game over. The player has to escape within 5 minutes. This project was a team effort. I was a game designer, level designer and programmer on the project.

Some notes about the project

  • Made in Unity's HDRP pipeline.
  • All 2D assets and animations were created by me.
  • This game won the game jam.
  • Most assets were made from the ground up with the exception being the skybox.
