For an assignment for University, I recreated an Mario Bros arcade in C++ using SDL 2

Some notes about the project

  • Made in C++.
  • Not 100% accurate to the Mario Bros Arcade game.
  • Can be played with both controller or keyboard
  • Can play with 2 players
  • Can load and play audio.
  • A 2D physics system was implemented, including callbacks and layers.
  • An ECS system was created for this project (sparse).
  • A custom texted based serializer was created.
  • A Editor with a moving camera and a prefab placement system.
  • Recreated most elements and gameplay mechanics from Mario Aracde including infinite levels..
  • Most enemies are programmed in the game.
  • There is a highscore table.
