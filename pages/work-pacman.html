Pacman

  • View Github page
  • View Video

For my first assignment in University, I created Pacman in C++. The aim was to try and recreate Pacman as close as I can while implementing my own ideas to the game.

Some notes about the project

  • Made in entirely in C++. A library was provided by the University to help input/output graphics and audio
  • Includes an ECS system that was programmed by me.
  • Includes a physics system that was programmed by me.
  • Ghost AI was recreated as close as possible to the original arcade game.
  • Infinite levels, with each level being more challanging than the previous.
  • This game has a local highscore table implemented.
