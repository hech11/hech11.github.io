For an assignment for University, I created an OpenGL sandbox application. The version of OpenGL used was 1.1.

Some notes about the project

  • Made in C++.
  • Includes an ECS system that was programmed by me.
  • Only OpenGL 1.1 was used to create everything.
  • Programmed an OBJ loader.
  • Skyboxes are implemented.
  • Dynamic lighting is programmed.
  • Can dynamically load any model, texture or material within the editor.
  • Can load .TTF files for fonts.
