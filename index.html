Pixelate

Pixelate is a game engine that is used to create 2D games. This project is still in development with the hopes of one day creating games with Pixelate. Currently Pixelate is being developed for desktop PC platforms with hopes to support other platforms in the future.

Some notes about the project

  • Made entirely in C++.
  • A batch renderer.
  • Audio system. Including spatial audio
  • Physics.
  • ECS system.
  • Can script entities via CSharp classes.
  • An editor. Including a scene and game viewer, hierarchy and component inspector and more.
  • Can save and load scenes.
View github repo

TV World

TV World was a college project where we had to simulate a small studio team. This was the first project I worked on where I managed and program a game to completion within a team.

Some notes about the project

  • Made in Unity using the HD render pipeline.
  • Code made, debugged and tested by me.
  • A save and load feature.
  • Controller support.
  • A dialogue system which allows designers to create a branching dialogue tree.
  • Used a hierarchical finite state machine(HFSM) to create the AI, scripted events and much more of the gameplay.
  • Minigames along with a online leaderboard for players to submit their highscores.
  • A shop to customise the main characters appearance.
View itch.io page
